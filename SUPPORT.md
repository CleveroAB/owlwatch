# Support

## Questions and deployment help

Use [GitHub Discussions](https://github.com/CleveroAB/owlwatch/discussions) for installation questions, deployment help, ideas, and general troubleshooting.

Before posting, check:

1. the [README](README.md), including configuration and GPU setup;
2. [Troubleshooting](docs/troubleshooting.md);
3. existing discussions and issues.

Include your owlwatch version, host OS and architecture, deployment method, relevant environment variable names (never secret values), and sanitized logs.

## Bugs

Use the [bug report form](https://github.com/CleveroAB/owlwatch/issues/new?template=bug_report.yml) for reproducible defects. Please keep feature requests in their dedicated form.

## Security issues

Do **not** discuss suspected vulnerabilities publicly. Follow [SECURITY.md](SECURITY.md).

## Scope

owlwatch is community-supported open-source software. Clevero AB does not currently offer an uptime SLA or private support contract for it.
